TBH ツール

Task Bar Hero（タスクバーヒーロー）の非公式ユーティリティ。ブックマークすればどの端末でも使えます。
⚡ 最強ビルドメーカー / Best Build
各部位の装備・宝石・彫刻・刻印をドロップダウンで選択してDPSを自動計算。「最強」ボタンで全クラス×全装備を総当りし、一番DPSが高い構成を自動セット。
💬 AIチャット / Ask Anything
アイテム・価格・ステージ・敵・クラス・スキルを横断でAIに相談。履歴保存・続きから会話。※ローカルAIサーバ起動が必要。
🔍 アイテム検索 / Item Search
装備・装飾(宝石)・彫刻・碑文(刻印)・特殊ステータスを効果や名前で検索。価格付き・日英対応。
🧱 クラフト素材 / Crafting
装備クラフト（キューブ）のレシピと必要素材を tier・部位で検索。素材ごとに Steam マーケットへ直リンク・価格付き。
⚡ 経験値効率 / EXP Farm
ヒーローレベルからオーバーレベル補正済みの経験値でステージを順位付け。クリアタイムを入れると毎時EXP（M/h）で並べ替え。
🛠 ビルドシミュレーター
装備・宝石・バフからDPSを計算。
📊 DPS計算機
シンプルなDPS計算。
データはコミュニティwiki由来でやや不確実、実機の値が優先。価格は tbh-market.com 経由（Steam相場）のスナップショット。
数値はリポジトリの tbh-data.json / tbh-prices.json を更新して再デプロイで反映。
